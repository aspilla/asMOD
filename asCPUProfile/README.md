#   asCPUProfile

## Introduction
An addon to check addon latency and performance.

## Displayed Information

*   **Addon Name**: The name of the addon.
*   **PeakTime**: The peak time the addon consumed CPU resources.
*   **BossAvg**: The average CPU usage during boss encounters.
*   **Over1ms, 5ms, 10ms, 50ms, 100ms**: The number of times the addon's CPU usage exceeded the specified duration. Spikes over 50ms can cause noticeable lag.
*   **OverSum**: A weighted sum of the "Over" values, used to rank addons by their overall CPU consumption.
![](https://media.forgecdn.net/attachments/1043/11/ascpuprofile.jpg)

## How to Use

*   **Activation**: The addon is disabled if its window is closed. You can reopen it after a combat encounter by typing the `/asCPU` command.

#   asCPUProfile

##  소개
애드온 Latency 성능 확인 

## 표시 내용

*   **애드온 이름:** 애드온의 이름.
*   **PeakTime:** 애드온이 CPU 리소스를 소비한 최대 시간,
*   **BossAvg:** 보스 전투 중 평균 CPU 사용량.
*   **Over1ms, 5ms, 10ms, 50ms, 100ms:** 애드온이 지정된 시간 이상 CPU 리소스를 소비한 횟수입니다. 50ms 이상은 렉을 유발.
*   **OverSum:** 1ms부터 100ms까지의 값을 가중 합산한 값으로, 전체 CPU 소비량을 기준으로 애드온 순위를 매기는 데 사용됨.
![](https://media.forgecdn.net/attachments/1043/11/ascpuprofile.jpg)

##  사용법

*   **활성화:** 창이 닫혀 있으면 애드온이 비활성화됩니다. 전투 후 `/asCPU` 명령어를 사용하여 다시 열 수 있음
