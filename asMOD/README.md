# asMOD Addon (Midnight)

Recommended WoW Default Settings and Edit Mode Configurations.

## Key Features

1. **Automatic asMOD Optimization Settings (`/asMOD`) (Recommended)**
    * When using the asMOD addon for the first time or by entering the `/asMOD` command, select `Settings` from the popup window.
    * **UI Scale adjusted to 0.75**
    * **Optimized Game Settings**: Always show nameplates, cooldown manager, damage meter display, etc.
    * **`asMOD Setup` Macro (Optional)**:
        * Provides useful settings such as world text size and camera rotation speed.
        * Check the macro via the `/m` command before use.
    * **Automatic Default UI Layout Configuration**:
        * Automatically imports and applies a predefined UI Edit Mode layout named "asMOD_layout".
        * When changing talents, you must select "asMOD_layout" in ESC > Edit Mode.

# asMOD 애드온 (한밤)

추천 와우 기본 설정 및 편집 모드 설정.

## 주요 기능

1.  **asMOD 최적화 설정 자동 적용 (`/asMOD`) (추천)**
    *   asMOD 애드온 최초 사용 시 또는 `/asMOD` 명령어로 실행 팝업창에서 `설정` 선택.
    *   **UI 스케일 0.75로 조정**
    *   **게임 설정 최적화** : 이름표 항상 표시, 쿨다운 메니저, 데미지 미터기 표시 등
    *   **`asMOD Setup` 매크로 (선택)**:
        *   월드 텍스트 크기, 회전 속도 등 유용한 설정.
        *   /m 명령어로 매크로 확인 후 사용 필요
    *   **기본 UI 레이아웃 자동 설정**:
        *   "asMOD_layout"이라는 이름으로 사전 정의된 UI 편집 모드 레이아웃을 자동으로 가져와 적용.
        *   특성 변경시 esc > 편집모드 에서 "asMOD_layout" 을 선택 필요.
