# asCountdown (Midnight)

Countdown Voice Alerts

## Key Features
* **Countdown Voice Alerts**: Plays voice announcements for 5, 4, 3, 2, 1 seconds only.
    
## Configuration
* No configuration options available.
* Voice files can be customized by replacing the `mp3` files inside the `sounds` folder (English) or `sounds_kr` folder (Korean).

# asCountdown (한밤)

초읽기 음성 알림

## 주요 기능
*  **초읽이 음성 알림**: 5초 4초 3초 2초 1초만 음성으로 알림    
    
## 설정
*  설정 없음
*  `sounds` 폴더 (영문), `sounds_kr` 폴더 (한글) 내 `mp3` 파일 변경으로 음성 변경 가능
