# asPowerBar (Midnight)

Displays the player's primary resources (Energy, Mana, Rage, etc.) and class resources (Combo Points, Runes, Holy Power, etc.).

![asPowerBar](https://github.com/aspilla/asMOD/blob/main/.Pictures/asPowerBar_MN.jpg?raw=true)

## Key Features

* **Primary Resource Bar**
    * Displays primary resources such as Mana, Rage, Energy, Runic Power, and Focus.
    * Includes **Resource Cost Prediction** during spell casting.
* **Class Resource Bar**
    * Displays class-specific resources in a segmented bar format.
    * Supports Combo Points, Holy Power, Soul Shards, Arcane Charges, Chi, Runes, and more.

## Supported Class Specializations & Resources

* **Evoker**: Essence
* **Paladin**: Holy Power
* **Mage**: [Arcane] Arcane Charges 
* **Warlock**: Soul Shards, [Destruction] Soul Shard Fragments
* **Druid**: Combo Points
* **Monk**: [Brewmaster] Stagger , [Windwalker] Chi
* **Rogue**: Combo Points, Super Charger
* **Death Knight**: Runes
* **Shaman**: [Elemental] Maelstrom , [Enhancement] Maelstrom Weapon 
* **Priest / Warrior / Demon Hunter / Hunter**: 

## Configuration
There are no separate settings.

*  **Positioning**:
    *   If the `asMOD` addon is installed, configuration is possible via the `/asConfig` command of `asMOD`.

# asPowerBar (한밤)

플레이어의 주 자원 (기력/마나/분노), 직업 자원(연계 점수, 룬, 신성한 힘 등) 표시.

![asPowerBar](https://github.com/aspilla/asMOD/blob/main/.Pictures/asPowerBar_MN.jpg?raw=true)

## 주요 기능 (Features)

*   **주 자원 바 (Primary Resource Bar):** 마나, 분노, 기력, 룬 마력, 집중 등의 주 자원을 표시. (시전 시 자원 소모 예측 기능 포함)
*   **직업 자원 바 (Secondary Resource Bar):** 연계 점수, 신성한 힘, 영혼의 조각, 비전 충전물, 기, 마력의 룬 등 직업별 보조 자원을 칸 형태로 표시.

## 직업 전문화 특성
*   **기원사 :** 정수
*   **성기사 :** 신성한 힘
*   **마법사 :** [비전] 비전 충전물
*   **흑마법사 :** 영혼의 조각, [파괴] 영혼의 조각 파편
*   **드루이드 :** 연계 점수
*   **수도사 :** [양조] 시간차, [풍운] 기.
*   **도적 :** 연계 점수, 초자력 충전 
*   **주술사 :** [정기]소용돌이, [고양]소용돌이치는 무기.
*   **죽음의 기사 :** 룬
*   **사제 :** 
*   **전사 :** 
*   **악마사냥꾼 :** 
*   **사냥꾼 :**
