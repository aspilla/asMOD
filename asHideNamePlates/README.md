# asHideNamePlates (Mid Night)

Hides nameplates except for mobs that are casting.

## Addon Activation Conditions

1. Adjustable in `ESC > Options > AddOns > asHideNameplates`.
2. `Key Binding` (Default): Hides nameplates while the hotkey is pressed. Hotkey can be set in `ESC > Options > Key Bindings > asMOD > asHideNamePlates Key`.
3. `Auto Trigger`: Automatically activates when a mob casts an interruptible spell.
4. `ALT + CTRL`: Activates when ALT and CTRL keys are pressed simultaneously.
5. `ALT`: Activates when ALT key is pressed.
6. `CTRL`: Activates when CTRL key is pressed.
7. `SHIFT`: Activates when SHIFT key is pressed.

## Nameplates that remain visible

1. Target or Focus target nameplates are always visible.
2. Casting mobs are visible.
3. For Tanks, nameplates of mobs where aggro is lost are visible.

## Supported Nameplates
1. Default Nameplates
2. asNamePlates
3. Plater Nameplates

# asHideNamePlates (한밤)

케스팅중인 몹 외 이름표를 숨김

## 애드온 동작 조건

1. `esc > 설정 > 애드온 > asHideNameplates` 에서 조정 가능
2. `Key Binding` (기본적 설정) : 단축키 누르고 있으면 이름표 숨김 기능이 동작. 단축키는 `esc > 옵션 > 단축키 설정 > asMOD > asHideNamePlates Key` 에서 지정 가능.
3. `Auto Trigger` : 몹이 차단 가능한 케스팅을 하면 자동으로 동작.
4. `ALT + CTRL` : ALT키 CTRL키를 동시에 누르면 동작.
5. `ALT` : ALT키를 누르면 동작.
6. `CTRL` : CTRL키를 누르면 동작.
7. `SHIFT` : SHIFT키를 누르면 동작.

## 숨겨지지 않고 표시 되는 이름표

1. 대상이나 주시 대상의 이름표는 표시.
2. 케스팅 중인 몹은 표시. 
3. 탱커의 경우 어그로를 놓친 몹의 이름표는 표시.

## 지원 이름표
1. 기본 이름표
2. asNamePlates
3. Plater Nameplates
