# asHideActionBar

Automatically hides action bars, which are not able to hide. Those are visible when you move mouse cursor near their original position.

## Target Frames
*   `MainActionBar` : Primary Action Bar
*   `StanceBar` : Stance Bar
*   `PetActionBar` : Pet Action Bar

**Note**: This addon does not provide an in-game configuration panel.
